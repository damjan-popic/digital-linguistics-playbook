# How do I make a tiny digital edition of a historical text?

<div class="answer-meta" markdown>
<span>Digital editions</span>
<span>beginner</span>
<span>60–120 min</span>
</div>

## What you are trying to do

You want to publish a small primary source with transcription, notes, and a link to the image.

!!! quote "One-sentence version"
    Start with a tiny sample, keep provenance, and only scale up once the workflow survives testing.

## You need

- Source image or scan
- Transcription policy
- Basic metadata

## Tools

- [TEI Guidelines](https://tei-c.org/guidelines/)
- [Markdown](https://www.markdownguide.org)

## Workflow

1. Describe the source before transcribing it.
2. Create a diplomatic transcription preserving original spelling and layout where relevant.
3. Add a normalized reading only if useful.
4. Annotate names, places, dates, abbreviations, and uncertain readings.
5. Publish image, transcription, notes, and citation together.

## Output

A small digital edition page.

## Check yourself

- Can readers distinguish source text from editorial intervention?
- Are uncertain readings marked?
- Is the source citable?

## Common traps

- Modernizing silently.
- Treating transcription as neutral copying.
- Publishing images without rights information.

## Practice task

Students transcribe the same short source and compare disagreements; the disagreements are the lesson.

## Useful extension

Turn the workflow into a small reusable checklist: input, command or tool action, output, quality check, and known failure mode.
