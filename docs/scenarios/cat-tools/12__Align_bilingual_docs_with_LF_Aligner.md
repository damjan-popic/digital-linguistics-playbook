**When to use** Create TM from document pairs.
**Input** `doc_en.txt` + `doc_es.txt`
**Tools** [LF Aligner](https://sourceforge.net/projects/aligner)
**Steps** Choose files ▸ Language pair ▸ Align
**Output** `aligned.tmx`
**Pro tips** Enable Google MT hints.
