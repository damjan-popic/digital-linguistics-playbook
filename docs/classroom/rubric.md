# Rubric

Use this for student contributions or peer review.

| Criterion | Excellent | Needs work |
|---|---|---|
| Question | One clear “How do I…?” problem | Topic is vague or tool-first |
| Workflow | Small, testable, ordered | Too broad, missing steps, or untested |
| Inputs/outputs | Concrete files or data types | Reader cannot tell what goes in or out |
| Checks | Gives ways to verify quality | Assumes success |
| Traps | Names real failure modes | Only gives generic warnings |
| Ethics/rights | Mentions privacy, copyright, provenance where relevant | Ignores reuse or harm |
| Style | Clear, friendly, concise | Dense, salesy, or unexplained jargon |
| Reproducibility | Another group can follow it | Depends on hidden knowledge |

## Minimal pass

A page is mergeable when another student can follow it without the author standing nearby. Harsh but fair. Also extremely educational.
