## What every page should give you

A useful scenario should make the task executable: what you need, what to run, what output to expect, how to check the result, and where the workflow can fail. The code examples do the same for public repositories, but by function rather than by repository nickname.

## Python 3.12 standard

The beginner NLP path uses **Python 3.12.x**. Do not jump to Python 3.13 or higher for those exercises unless a specific project says it has been tested there.
