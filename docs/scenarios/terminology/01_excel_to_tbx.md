**When to use** Need a standards-compliant TBX from a spreadsheet.  
**Input** `glossary.xlsx` (columns: ID, EN, ES, Definition)  
**Tools** [Glossary Converter](https://www.glossaryconverter.com)

**Steps**

1. Drag file onto Glossary Converter.  
2. Map columns.  
3. Select **TBX-Basic** → **Convert**.

**Output** `glossary.tbx`

!!! tip "Pro tip"
    Tick **Generate term IDs** if your sheet lacks them.
