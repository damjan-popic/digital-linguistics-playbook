# Peer review

Peer review should be kind, specific, and merciless about ambiguity.

## Reviewer checklist

- What is the one question this page answers?
- Did you know what input to start with?
- Did any step require hidden knowledge?
- Did the output match the promised output?
- What broke?
- What ethical, legal, or privacy issue is missing?
- Which sentence was most useful?
- Which sentence should be deleted?

## Review format

```text
What worked:
What broke:
Most confusing step:
Missing check:
Suggested fix:
Would I merge this? yes/no/not yet
```

## Rule of thumb

Do not only correct the page. Improve the workflow. A beautiful tutorial for a broken process is still a tiny disaster in a nice jacket.
