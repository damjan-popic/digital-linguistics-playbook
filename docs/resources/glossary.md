# Glossary

**CAT tool**  
Computer-assisted translation tool. It usually supports translation memories, termbases, QA, segmentation, and project management.

**Corpus**  
A collection of texts assembled for analysis. A corpus is not just “some files”; it has scope, composition, metadata, and limits.

**FAIR**  
A shorthand for data that is findable, accessible, interoperable, and reusable.

**Gazetteer**  
A place-name index, often with coordinates and alternative names.

**KWIC**  
Key Word in Context. Concordance lines showing a search hit with surrounding text.

**Metadata**  
Data about data: source, creator, date, rights, language, processing history, uncertainty, and so on.

**OCR**  
Optical character recognition: extracting text from images or scanned documents.

**Provenance**  
The record of where data came from and how it was transformed.

**TBX**  
TermBase eXchange, an XML-based format for terminology data.

**TEI**  
Text Encoding Initiative: guidelines for representing texts, especially scholarly and historical materials, in XML.
